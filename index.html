Sobre mi

Mi nombre es Josshua Nolasco y soy Data Scientist. Tengo experiencia de machine learning y de algunos algoritmos de deep learning he trabajado en empresas privadas y centros de investigación.

Me aseguro de utilizar siempre las últimas tecnologías, principalmente open source como Python y R, además, SAS, MatLab, STATA, E-Views, SSPS, SQL Server y Spark. Si en tu empresa tienes datos y necesitas potenciarla y llevarla al siguiente nivel te aseguro que llegaste al lugar correcto.

Servicios

Data Wrangling

Proceso de mapear datos de un formato "crudo" a otro formato que permite un manejo más eficiente de los datos con ayuda de herramientas semiautomatizadas y/ó hasta la creación de una automatizada para cierta tarea específica.

Data Mining

Conjunto de técnicas y heramientas que permiten explorar grandes bases de datos, de manera semiautomática o semiautomática, con el objetivo de encontrar patrones, tendencias o relgas que expliquen el comportamiento de los datos.

Machine Learning

Estudio científico de algoritmos y modelos estadísticos que utilizan sistemas informáticos para realizar una tarea específica como agrupación, predicción, detección de anomalías, reglas de asociación, árboles de desicion,...

Deep Learning

Modelos de redes neuronales que se aplican a grandes volúmenes de datos para el descubrimiento y aplicación del conocimiento adquirido para realizar predicciones apartir de él.

Optimización

Técnicas de optimizaciín lineal y no lineal de procesos con herramientas de cálculo que se basan en algoritmos matematicos.

Simulación de Montecarlo

Metodos estocasticos que permiten simular un proceso y predecir el valor óptimo que permita eficientar cualquier proceso sin solución exacta.
